# 🛠️ DevToolkit

![Python Version](https://img.shields.io/badge/python-3.10%2B-blue.svg)
![UI Library](https://img.shields.io/badge/UI-Rich-green.svg)
![License](https://img.shields.io/badge/license-MIT-orange.svg)

**DevToolkit** is a modern, terminal-based utility suite built for developers and everyday tech tasks. It combines essential tools into a single, highly interactive CLI interface powered by Python and the `rich` library.

---

## ✨ Features

- 🔐 **Password Generator**: Generate strong, secure, and customizable passwords.
- 🧮 **Calculator**: Perform quick, error-handled arithmetic calculations.
- 🔒 **Hash Generator**: Generate secure MD5, SHA-256, and SHA-512 hashes from input strings.
- 📱 **QR Code Generator**: Create customized QR codes and save them directly to the `outputs/` directory.
- 📄 **JSON Formatter**: Format and validate raw JSON strings into clean, readable syntax with color highlighting.
